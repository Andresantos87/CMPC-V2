# **App Name**: Digital PTW

## Core Features:

- Create Work Permit: Form to fill out necessary fields for a new digital work permit.
- View Permits: Browse all existing released permits.
- Database storage: Storage of the data filled by the form for future releases, editions and consults.
- Edit Released Permits: Makes it possible to edit existing work permits.

## Style Guidelines:

- Primary color: Dark Blue (#3F51B5) for a sense of trust and security.
- Background color: Light Gray (#F0F2F5) for a clean and professional look.
- Accent color: Orange (#FF9800) for CTAs and important actions.
- Body and headline font: 'Inter', sans-serif, for a modern and readable design.
- Use clear and simple icons for main actions like 'Create' and 'Edit'.
- Use a consistent grid layout to present the information in a structured manner.
- Subtle animations on button interactions.
